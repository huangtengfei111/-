![03_缓存穿透现象以及解决方案](E:\U盘\视频\26_你能说说我们一般如何应对缓存雪崩以及穿透问题吗？\03_缓存穿透现象以及解决方案.png)

```
缓冲穿透
 恶意攻击，请求的数据在数据库中压根不存在的数据，导致每次请求过来时缓存中没有，都回去数据库查询，导致数据库压力大宕机
 措施：
 当数据库查不到时，在缓存中存入这个key,value设为unknown，这样下次请求过来时就不会落到数据库中
```
